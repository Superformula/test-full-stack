# Superformula User App

##### Project setup
```
yarn install
```

##### Compiles and hot-reloads for development
```
yarn serve
```

##### Compiles and minifies for production
```
yarn build
```

##### Lints and fixes files
```
yarn lint
```

##### Unit tests
```
yarn test
```

##### E2E tests
```
cypress run
```

### Deployment

##### Initial S3 Bucket creation
```
yarn sam:createBucket
```

##### Application deployment
```
yarn push
```
